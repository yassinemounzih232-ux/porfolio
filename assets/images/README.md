Placez ici vos images de projet avec les noms suivants :

- `rapportup.jpg`  (image de présentation RapportUp)
- `evup.jpg`       (image de présentation EvUp)

Formats supportés : .jpg, .png, .webp
Taille recommandée : environ 1200x800 px pour un rendu optimal.

Après avoir ajouté les fichiers, rechargez la page `acceuil.html` sur le serveur local pour voir les images s'afficher.
